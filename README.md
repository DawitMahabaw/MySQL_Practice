# MySQL Practice 🚀

> A hands-on Node.js and MySQL project demonstrating database connectivity, SQL table creation, and data insertion using Express.

---

## 📌 Overview

This project contains my solutions to a series of MySQL practice exercises completed while learning backend web development.

The project demonstrates how to:

- Connect a Node.js application to a MySQL database
- Create databases and tables using SQL
- Build an Express server
- Handle HTTP requests
- Insert data into a MySQL database from an HTML form

---

## ✨ Features

- 🔗 Connect Node.js to a MySQL database
- 🗄️ Create database tables using SQL
- 🌐 Build an Express server
- 📝 Handle HTML form submissions
- 📥 Insert records into MySQL
- 📦 Organize backend code using Node.js modules

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/DawitMahabaw/MySQL_Practice.git
```

Navigate into the project

```bash
cd MySQL_Practice
```

Install dependencies

```bash
npm install
```

---

## 🚀 Running the Project

Start the application

```bash
node app.js
```

Then open your browser and visit

```text
http://localhost:3001/
```

or

```text
http://localhost:3001/install
```

depending on the exercise you're testing.
